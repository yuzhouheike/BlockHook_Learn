Q : 企安殿学员在APP端都能做什么？
A :
企安殿学员在APP端可以学习i春秋线上所有课程，同时可以在APP首页学习管理员设置的必修课和选修课，在课程库学习职业课程和行业课程。企安殿学员还可以在个人中心查看考试记录，查询历史考试成绩和排名。
Q : 企安殿学员如何查看必修课和选修课？
A :
企安殿管理员为学员设定完必修课/选修课后，会通过系统消息提醒学员，学员可在APP首页查看必修课和选修课。
Q : 企安殿学员如何进行随堂测验？
A :
APP端目前暂不支持随堂测验，如需随堂测验请登录企安殿官网进行操作。
Q : 企安殿学员如何进行课程测验？
A :
APP端目前暂不支持课程测验，如需课程测验请登录企安殿官网进行操作。
Q : 企安殿学员如何参加管理员设定的考试？
A :
登录企安殿官网，进入考试模块可查看和参加管理员设置的考试，在企安殿官网考试模块和APP上均可查看考试成绩和排名。
